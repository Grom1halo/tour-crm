# Tour CRM — Чеклист перед переносом на новый домен

## 1. Инфраструктура
- [ ] Новый домен/IP настроен и резолвится
- [ ] Nginx конфиг: proxy_pass на backend (порт 3000), статика frontend из `/dist`
- [ ] SSL сертификат (если нужен)
- [ ] PostgreSQL запущен, база `tour_crm` доступна
- [ ] PM2 процесс `tour-crm` запущен (`pm2 status`)
- [ ] Backend отвечает: `GET /api/auth/me` → 401 (не 502/404)
- [ ] Frontend загружается (открыть / в браузере)

## 2. Авторизация
- [ ] Страница /login отображается
- [ ] Вход с правильным логином/паролем → редирект на главную
- [ ] Неверный пароль → сообщение об ошибке
- [ ] Выход из системы работает
- [ ] Неавторизованный запрос → редирект на /login
- [ ] Токен сохраняется в localStorage, не теряется при F5

## 3. Ваучеры
- [ ] Список ваучеров загружается
- [ ] Поиск по номеру/клиенту работает
- [ ] Фильтр по дате, статусу, менеджеру работает
- [ ] Создание нового ваучера (все поля сохраняются)
- [ ] Выбор компании → фильтруются туры этой компании
- [ ] Выбор тура → автозаполняется компания
- [ ] Выбор тура+компании+даты → автозаполняются нетто-цены
- [ ] Редактирование ваучера
- [ ] Просмотр ваучера (страница детали)
- [ ] Добавление платежа в ваучере
- [ ] Копирование ваучера
- [ ] Удаление ваучера (soft delete)
- [ ] Восстановление удалённого ваучера
- [ ] Кнопка "Новый ваучер для клиента" → клиент предзаполнен

## 4. Хотлайн
- [ ] Страница /hotline загружается
- [ ] Показываются ваучеры на сегодня

- [ ] Колонка "Клиент" отображается
- [ ] Кнопка "↓ Excel" скачивает файл за выбранный день
- [ ] Сортировка по колонкам работает

- [ ] Переключение группировки: менеджер / компания / тур / день

- [ ] Фильтр "По продаже" / "По выезду" работает
- [ ] Вкладка "Платежи" — список платежей
- [ ] Вкладка "Детально" — таблица ваучеров
- [ ] Блок зарплат менеджеров (для admin/accountant)

- [ ] HTML отчёт открывается в новой вкладке
- [ ] Фильтр по менеджеру (для admin/accountant)

## 6. Справочники
- [ ] Компании: список, создание, редактирование

- [ ] Агенты: список, создание, редактирование, удаление
- [ ] Менеджеры: список пользователей
- [ ] Клиенты: список (только admin)

## 7. Роли доступа
- [ ] **admin** — доступ ко всему
- [ ] **manager** — ваучеры (своих клиентов), отчёты (свои данные), хотлайн, агенты
- [ ] **accountant** — просмотр ваучеров, отчёты (все), менеджеры
- [ ] **hotline** — только /hotline
- [ ] **editor** — только /companies и /tours
